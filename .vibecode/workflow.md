# Workflow

## Standard Loop

```text
1. Read specs.
2. Plan.
3. Implement minimal phase.
4. Test.
5. Update docs.
6. Review diff.
```

## Use These Commands

```text
/01-plan
/02-implement
/03-review
/04-security-pass
```

## Do Not

```text
- Mix phases
- Invent paid infra
- Skip env examples
- Skip manual tests
```
